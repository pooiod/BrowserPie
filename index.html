BrowserPie

Run Python from your web browser!

What is BrowserPie?

BrowserPie brings Python to your web browser, letting websites do more than the sandbox allows for. Once installed, you'll be able to test and execute Python scripts from any webpage.

Get Started with BrowserPie

To use BrowserPie, you need Python installed. Follow the steps below:

  • 1. Install Python if you don't have it.
  • 2. Download BrowserPie for Windows.
  • 3. Change some settings and start using Python in your browser!

For instructions on how to program a website to use Python with BrowserPie, check out the documentation.
